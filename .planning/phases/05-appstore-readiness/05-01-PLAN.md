# Phase 05: iOS 上线合规 + 终审修复 — PLAN.md

## 目标

修复 App Store 4 项阻塞问题 + 验收审查 Medium 级问题，使项目达到可发布状态。

## Wave 1: App Store 阻塞项（必须修复）

### Task 5.1: ITSAppUsesNonExemptEncryption

**问题:** App 使用 WSS + HMAC-SHA256，缺少加密豁免声明。Apple 必拒。

**方案:**
1. 在 `ios/Runner/Info.plist` 中添加 `<key>ITSAppUsesNonExemptEncryption</key><false/>`
2. 添加 `NSAppTransportSecurity` 允许本地网络连接

**文件:** `ios/Runner/Info.plist`

### Task 5.2: PrivacyInfo 数据采集声明

**问题:** `device_info_plus` 采集 IDFV，但 PrivacyInfo 声明空数据。必拒。

**方案:**
1. 在 `PrivacyInfo.xcprivacy` 中声明 `NSPrivacyCollectedDataTypeDeviceID`
2. 用途：AppFunctionality，非追踪，非关联

**文件:** `ios/Runner/PrivacyInfo.xcprivacy`

### Task 5.3: 移动端音频实现

**问题:** `flame_audio_impl.dart` 仅为 debugPrint stub。iOS/Android 完全静音。

**方案:**
1. 使用 `flame_audio` 包加载 `assets/sounds/*.wav`
2. 实现真实的 `playHit()`/`playSkill()`/`playJump()` 等
3. 保留 Web Audio API 路径不变

**文件:** `lib/game/audio/flame_audio_impl.dart`, `lib/game/audio/sound_manager.dart`

## Wave 2: 终审修复（提升质量）

### Task 5.4: 英雄数值微调

**问题:** 荆轲 52攻过高（同样 195速的貂蝉仅 38攻）。Hou Yi/Jing Ke/Shaolin Monk 技能伤害硬编码。

**方案:**
1. 荆轲 attackPower 52→46
2. `hou_yi.dart`: damage 250→skillDamage
3. `shaolin_monk.dart`: 两处硬编码伤害→skillDamage
4. `jingke.dart`: damage 90→skillDamage

**文件:** `jingke.dart`, `hou_yi.dart`, `shaolin_monk.dart`

### Task 5.5: 测试更新

**问题:** 9 个测试因数值重平衡失败（吕布 250→240 等）。

**方案:**
1. `hero_data_test.dart`: 更新期望值匹配新数值
2. `combat_system_test.dart`: 更新技能伤害期望
3. 目标：224/224 测试全部通过

**文件:** `test/hero_data_test.dart`, `test/combat_system_test.dart`

### Task 5.6: AI 跳跃 + 音效开关

**问题:** AI 从不跳跃；设置音效开关未连接 SoundManager。

**方案:**
1. `ai_controller.dart`: 在 `dodgeUp` 和躲避弹体时设置 `fighter.input.jump = true`
2. `main_menu.dart`: 音效开关连接 `SoundManager().setEnabled(bool)`
3. `sound_manager.dart`: 添加 `setEnabled()` 方法

**文件:** `ai_controller.dart`, `main_menu.dart`, `sound_manager.dart`

### Task 5.7: 元数据更新

**问题:** pubspec.yaml 描述为占位符，web/index.html 标题为 hero_fighter。

**方案:**
1. `pubspec.yaml`: description 改为 "A 2D fighting game with 12 heroes and online multiplayer"
2. `web/index.html`: title→"Hero Fighter", meta description 更新
3. `app_config.dart`: 移除硬编码 secret 注释中的默认值暴露

**文件:** `pubspec.yaml`, `web/index.html`, `lib/config/app_config.dart`
