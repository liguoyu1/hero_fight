Hero Fighter Support / 技术支持

FAQ

Q: How to play online? / 如何在线对战？

Select "Online Battle" from the main menu. The game will automatically match you with another player.

从主菜单选择「在线对战」，游戏会自动为你匹配对手。

Q: How to play with friends on same WiFi? / 如何和朋友局域网对战？

Select "Local 2P" mode. Both players use the same device — P1 uses keyboard (WASD + J/K), P2 uses Arrow keys + Numpad 1/2.

选择「本地双人」模式。两名玩家使用同一设备对战。

Q: The game has no sound / 游戏没有声音？

Check the settings menu (gear icon on main menu) to ensure sound effects are enabled. On iOS, make sure silent mode is off.

检查主菜单设置中的音效开关。在 iOS 上请确保静音模式已关闭。
